# Da ELIZA aos Transformers
Este projeto demonstra a evolução histórica dos modelos de linguagem natural, desde sistemas baseados em regras até modelos modernos baseados em redes neurais.
## Objetivo
Compreender a transição entre:
- Sistemas simbólicos (ELIZA)
- Modelos estatísticos (N-Grams)
- Modelos neurais (Transformers)
